## рҹ“Ў ProtokГіЕӮ Modbus RTU over RS232

**Baud:** 9600 | **Parity:** None | **Stop bits:** 1 | **Address:** 0x01

| Rejestr | Typ | Opis |
|---------|-----|------|
| `0x0001` | Float | NapiДҷcie zadane [V] |
| `0x0003` | Float | PrД…d zadany [A] |
| `0x001D` | Float | NapiДҷcie aktualne [V] |
| `0x001F` | Float | PrД…d aktualny [A] |
